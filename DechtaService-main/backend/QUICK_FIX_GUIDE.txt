Q: PostgreSQL connection error?
A: Check .env file has correct settings:
   DB_HOST=localhost
   DB_PORT=5432
   DB_USER=postgres
   DB_PASSWORD=postgres
   DB_NAME=dechta

Q: "Table already exists" error?
A: That's OK! It means table was already created. Fix-all.js handles this.

Q: "Column already exists" error?
A: That's OK! Same reason - the script is idempotent and safe to run again.

Q: Still getting 500 errors after fix-all.js?
A: 1. Verify fix-all.js showed "SUCCESS!" message
   2. Check backend restarted: npm run dev
   3. Check code file was updated:
      grep "otp_verified" src/controllers/ordersController.js
   4. Run quick-check.js to verify columns exist

Q: Backend not starting after npm run dev?
A: 1. Stop any running instance: Ctrl+C
   2. Try again: npm run dev
   3. Check error logs for missing dependencies
   4. Run: npm install (to reinstall)

Q: Want to see all data in database?
A: Run: node generate-report.js
   This creates DATABASE_REPORT.txt with full analysis
